
Dckerfile for Android Development Environment
===========

 * OpenJDK 8
 * Android SDK r22.2
 * build-tools-22.0.1

### Docker pull

    docker pull hotchemi/android-22

### Run

    docker run -i -t hotchemi/android-22 /bin/bash

## Android SDK

    Version : r22.2
    Install Location : /usr/local/android-sdk-linux
